# Install Linux OS

## Installasi

Install Ubuntu LTS (19.04, 20.04 , 21.04)
```bash
boot dari cd
```

Install PostgreSQL
```bash
$ sudo apt-get install postgresql-12
```


Install Linux Management & Support Tools

```bash
$ sudo apt-get install htop mc lynx
$ sudo apt-get install openssh-server 
```
